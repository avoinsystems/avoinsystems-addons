Finnish Invoice, Delivery Date

What's in it?

This module adds delivery date -field to the invoice, as is required by the Finnish accounting law.

This module does not alter the existing invoice prints, but rather this fields is mainly be used by the Finnish invoice template.

Tämä moduuli lisää laskulle Suomen kirjanpitolain mukaisen toimituspäivämäärä -kentän.

Tämä moduuli ei muokkaa olemassaolevia laskupohjia, vaan ensisijaisesti tätä kenttää käytetään suomalaisella laskupohjalla.
